### AbsenQ

ini adalah aplikasi saya yang di bangun pada tahun 2019, ketika saya duduk di kelas 12 smkn 1 kawali.
Aplikasi ini saya berinama AbsenQ. aplikasi ini dibangun menggunakan framework laravel v6.8 dan database mysql lengkap dengan perencanaan sistem.
